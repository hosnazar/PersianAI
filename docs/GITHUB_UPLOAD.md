# GitHub Upload Guide

## Option 1: GitHub Web Upload

1. Go to GitHub
2. Create a new repository
3. Name it: iranian-ai-assistant
4. Keep it public or private
5. Upload all files from this folder
6. Commit changes

## Option 2: Git CLI

```bash
git init
git add .
git commit -m "Initial Persian AI assistant MVP"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/iranian-ai-assistant.git
git push -u origin main
```
